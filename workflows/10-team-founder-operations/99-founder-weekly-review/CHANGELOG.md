# Changelog — Founder Weekly Review

## [0.1.0] — 2026-07-14

### Added
- Initial release (beta)
- Weekly Schedule trigger (Friday 17:00) + Manual Trigger
- Review data: wins, struggles, decisions (confidence + reversibility), 4 key metrics with week-over-week comparison
- LLM at temperature 0.5: week headline, key pattern, leverage point, metric commentary, decision quality, energy note, next-week intention, coaching question
- JSON validation with error routing
- Sent directly to founder — no approval gate needed (personal insight)

### Known issues at release
- Not runtime-tested
- Review data requires manual entry — add metric API calls for full automation
- LLM has no access to previous weeks' reviews — no longitudinal pattern analysis
