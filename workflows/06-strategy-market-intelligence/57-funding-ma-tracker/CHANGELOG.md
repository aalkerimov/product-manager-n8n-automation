# Changelog — Funding and M&A Tracker

## [0.1.0] — 2026-07-14

### Added
- Initial release (beta)
- Weekly Tuesday schedule trigger
- Manual Trigger with 4 embedded sample events (2 funding, 1 acquisition, 1 key hire)
- LLM prompt for positioning implications analysis per event
- Aggregate signal synthesis and biggest risk/opportunity identification
- JSON validation with error routing
- Human approval gate with 48h timeout
- Rejection logging

- No real news/RSS feed reading — must be added in production
