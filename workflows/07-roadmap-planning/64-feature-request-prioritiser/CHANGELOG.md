# Changelog — Feature Request Prioritiser

## [0.1.0] — 2026-07-14

### Added
- Initial release (beta)
- Manual Trigger with 20 embedded sample feature requests
- LLM theme clustering with vote aggregation per cluster
- Top-10 ranked shortlist with impact, strategic fit, and rationale per item
- Quick-wins list and defer list
- JSON validation with error routing
- Human approval gate with 48h timeout
- Rejection logging

### Known issues at release
- Not runtime-tested
- No integration with Productboard, Jira, or Intercom out of the box
- Vote equality across segments — enterprise votes not weighted higher by default
